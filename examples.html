jQuery SpanSlicer

Basic Usage

The easiest way to instantiate a new SpanSlicer is simply to call .spanslicer() on the wrapped set:

      <div id="spanner_1"></div>
      <input type="button" name="" value="Split selected" id="split_button_1"/> <input type="button" name="" value="Delete selected" id="delete_button_1"/>
      <script type="text/javascript" charset="utf-8">
        $(function() {
          $("#spanner_1").spanslicer();
          $("#split_button_1").bind('click',function(event) {
            $.spanslicer.splitSpan("#spanner_1");
          });
          $("#split_button_1").bind('click',function(event) {
            $.spanslicer.deleteSpan("#spanner_1");
          });
        });
      </script>

By default, the SpanSlicer starts with 96 divisions, with a tickmark every 4 divisions (this works great for diving a 24-hour day into 15-minute segments)

To instantiate a SpanSlicer with pre-existing slices, you can either (a): Create a series of divs with a "data-offset" attribute, being a number indicating the index of the slicepoint:

      <div id="spanner_2">
        <div data-offset="0"></div>
        <div data-offset="23"></div>
        <div data-offset="25"></div>
        <div data-offset="45"></div>
        <div data-offset="78"></div>
      </div>
      <input type="button" name="" value="Split selected" id="split_button_2"/> <input type="button" name="" value="Delete selected" id="delete_button_2"/>
      <script type="text/javascript" charset="utf-8">
        $(function() {
          $("#spanner_2").spanslicer();
          $("#split_button_2").bind('click',$.spanslicer.reference("#spanner_2").splitSpan);
          $("#delete_button_2").bind('click',$.spanslicer.reference("#spanner_2").deleteSpan);
        });
      </script>

...or (b): Simply pass an array of the slicepoint indices as the "spans" option to the $.fn.spanslicer() function:

      <div id="spanner_3"></div>
      <input type="button" name="" value="Split selected" id="split_button_3"/> <input type="button" name="" value="Delete selected" id="delete_button_3"/>
      <script type="text/javascript" charset="utf-8">
        $(function() {
          $("#spanner_3").spanslicer({
            spans: [0,23,25,45,78]
          });
          $("#split_button_3").bind('click',$.spanslicer.reference("#spanner_3").splitSpan);
          $("#delete_button_3").bind('click',$.spanslicer.reference("#spanner_3").deleteSpan);
        });
      </script>
